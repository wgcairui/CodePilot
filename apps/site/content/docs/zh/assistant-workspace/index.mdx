---
title: 助手工作区
description: 检查和管理 Claude 的工作环境。
---

助手工作区让你能够查看 Claude 在对话过程中的操作。它展示文件变更、工具使用情况以及 Claude 当前的工作上下文状态。

## 功能

### 文件检查器

查看 Claude 在对话中读取或修改的文件：

- **文件树** — 浏览项目目录结构
- **Diff 视图** — 查看 Claude 对每个文件的具体更改
- **文件预览** — 带语法高亮的文件预览

### 工具活动

实时监控 Claude 的工具使用：

- 查看 Claude 正在调用哪些工具及原因
- 查看工具输入/输出以便调试
- 追踪操作序列

### 上下文使用情况

追踪 Claude 上下文窗口的使用情况：

- **Token 计数** — 当前使用的 token 数 vs. 可用数
- **使用明细** — token 在消息、工具和系统上下文中的分布
- **成本估算** — 基于当前提供商定价的大致费用

## 打开工作区

工作区面板可以通过以下方式切换：

- 前往**设置**，打开**助手**部分（即 Settings #assistant）
- 键盘快捷键（可在设置中配置）

## 使用场景

- **代码审查** — 在提交前审查 Claude 所做的所有文件更改
- **调试** — 当某些功能不正常时检查工具调用
- **学习** — 通过观察工具使用来了解 Claude 如何处理问题
- **成本管理** — 监控 token 使用以控制预算
