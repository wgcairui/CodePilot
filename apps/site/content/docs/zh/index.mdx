# 快速开始

欢迎使用 CodePilot！本指南将帮助你快速上手。

## 什么是 CodePilot？

CodePilot 是 Claude Code 的原生桌面 GUI 客户端，基于 Electron 和 Next.js 构建。
它提供了丰富的图形界面，用于与 Claude 交互、管理项目以及利用 AI 驱动的开发工具。

## 快速上手

1. 从[下载页面](/zh/download)下载 CodePilot
2. 安装并启动应用
3. 在设置中配置你的 API 提供商
4. 开始新对话，与 Claude 一起编程！

## 功能特色

- **多提供商支持** — Anthropic、OpenAI、Google、AWS Bedrock
- **MCP 插件系统** — 通过 Model Context Protocol 插件扩展功能
- **消息桥接** — 连接 Discord、Telegram、飞书和 QQ
- **助手工作区** — 检查和管理 Claude 的工作环境
- **主题系统** — 可自定义的主题和视觉风格
- **图片生成** — 内置 AI 图片生成功能
