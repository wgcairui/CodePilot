# Getting Started

Welcome to CodePilot! This guide will help you get up and running.

## What is CodePilot?

CodePilot is a native desktop GUI client for Claude Code, built with Electron and Next.js.
It provides a rich graphical interface for interacting with Claude, managing projects,
and leveraging AI-powered development tools.

## Quick Start

1. Download CodePilot from the [download page](/download)
2. Install and launch the application
3. Configure your API provider in Settings
4. Start a new conversation and begin coding with Claude!

## Features

- **Multi-provider support** — Anthropic, OpenAI, Google, AWS Bedrock
- **MCP Plugin System** — Extend functionality with Model Context Protocol plugins
- **Message Bridge** — Connect to Discord, Telegram, Feishu, and QQ
- **Assistant Workspace** — Inspect and manage Claude's working environment
- **Theme System** — Customizable themes and visual styles
- **Image Generation** — Built-in AI image generation capabilities
